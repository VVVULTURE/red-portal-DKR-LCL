⚡ Local Proxy Server Config

This build runs its proxy locally via server.js — no Cloudflare account needed.

  1. Make sure the server is running: node server.js
  2. The default proxy is https://red-portal-service.koyeb.app/proxy
  3. If you used a custom port (e.g. PORT=8080 node server.js), update the URL below.
